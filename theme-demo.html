Features

  • Dynamic Icons: Moon icon in light mode, sun icon in dark mode
  • Persistent Storage: Your preference is saved in localStorage
  • Auto-Apply: Theme applies automatically on page load
  • System Preference: Respects system dark mode if no preference set
  • Smooth Transitions: Elegant animations when switching themes
  • High Contrast: Optimized colors for readability in both modes

How It Works

The theme toggle is located in the navigation bar, to the right of the "Contact" button. Click it to toggle between Day and Night modes:

  • Day Mode (Light): Uses light background with dark text, moon icon
  • Night Mode (Dark): Uses dark background with light text, sun icon
üí° Tip: The icon shows what clicking will activate. In dark mode, the sun icon means clicking switches to light mode, and vice versa.

Technical Details

CSS Classes:

  • .colorscheme-light - Light theme styles
  • .colorscheme-dark - Dark theme styles

Icons:

  • fa-moon - Displayed in light mode
  • fa-sun - Displayed in dark mode (golden color)

Storage:

  • Key: colorscheme
  • Values: "light" or "dark"
